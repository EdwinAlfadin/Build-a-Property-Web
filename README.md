## рҹ“– Overview

My House Resident is a modern real estate website designed to showcase residential properties with a clean, responsive, and user-friendly interface.

This project was built to practice front-end development by creating a multi-page website that focuses on responsive layouts, reusable components, and modern UI design.

The website provides visitors with property information, house types, residential amenities, blog articles, and a contact page, offering a complete browsing experience across desktop, tablet, and mobile devices.
